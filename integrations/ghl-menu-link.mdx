## Overview

Import sub-accounts and install Assistable as a custom menu link in your GoHighLevel workspace. This gives your clients direct access to their AI dashboard from within GHL.

## Setup

<Steps>
  <Step title="Navigate to Settings">
    In your Assistable workspace, go to the integration settings.
  </Step>
  <Step title="Copy the Menu Link URL">
    Copy the custom menu link URL provided for your workspace.
  </Step>
  <Step title="Install in GHL">
    In GoHighLevel, go to Settings → Custom Menu Links → Add the Assistable URL as a new menu item.
  </Step>
  <Step title="Configure Access">
    Set which sub-accounts can see the menu link and configure any access restrictions.
  </Step>
</Steps>
