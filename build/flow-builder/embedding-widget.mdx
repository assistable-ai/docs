## Overview

Embed your flow agent as a chat widget on any website.

## Configuration

Configure this component in the Flow Builder's visual editor. Drag it onto the canvas and connect it to other nodes to build your conversation flow.

<Tip>Use the built-in testing interface to verify each node works correctly before connecting it to the broader flow.</Tip>
