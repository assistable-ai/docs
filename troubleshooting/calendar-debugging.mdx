## Common Calendar Issues

### Calendar Not Synced
- Go to Calendar settings and verify the correct calendar is selected
- Re-sync the connection if needed
- Test by running a calendar availability check

### Wrong Availability Showing
- Check calendar permissions
- Verify timezone settings match between your calendar and the assistant
- Ensure no conflicts with other calendar tools

### Bookings Not Appearing
- Confirm all calendar tools are enabled (Check Availability, Book, Reschedule, Cancel)
- Test each tool individually
- Check the calendar for permission restrictions
