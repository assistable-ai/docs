## Overview

Use automatic data collection to populate custom fields in GoHighLevel without manual entry. As the AI collects information during conversations, it can automatically update the contact's custom fields in real-time.

## Setup

1. Create the custom fields in GHL Settings
2. Configure the data collection rules in your assistant
3. Map collected data to the corresponding custom fields
4. Test with a conversation to verify fields are populated
